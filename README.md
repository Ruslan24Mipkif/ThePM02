
# Дневник производственной практики

Простая программа для автоматизации учета выполненных работ во время прохождения производственной практики. Разработана в рамках модуля ПМ.02

## Функционал
- [X] Добавление записей о практике.
- [X] Просмотр записей в удобном UI
- [X] Удаление ошибочных или устаревших записей.
- [X] Авто расчет общего количества отработанных часов.
- [X] Сохранение данных в txt файл.

## Технологический стек
- Python 3
- Интерфейс: Tkinter/ttk
- Хранение данных: diary_data.txt

## Установка и запуск
1. Склонируйте репозиторий или скачайте файлы.
2. Установите Python 3.10
3. Запустите основной модуль:
   ```bash
   python main.py
